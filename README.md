# 🏡 Real Estate Web Application

A modern and responsive real-estate web platform built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
The application allows users to explore property listings with smooth animations and seamless navigation.

---

## 🚀 Features

- Dynamic property listings  
- RESTful API integration  
- Scalable MongoDB database architecture  
- Smooth animations using **GSAP**  
- Interactive UI transitions with **Framer Motion**  
- Fully responsive design  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- JavaScript  
- GSAP  
- Framer Motion  

### Backend
- Node.js  
- Express.js

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

## 2️⃣ Install Dependencies
```bash
cd client
npm install

cd ../server
npm install
```

## 3️⃣ Configure Environment Variables

Create a .env file in the server folder:

```bash
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

## 4️⃣ Run the Application
# Start backend
```bash
npm run server
```
# Start frontend
```bash
npm start
```
